# geometry game

A simple geometry game where the player attempts to guess both the area,
and a point which falls inside a randomly generated rectangle.

```
$ python3 run.py

Rectangle coordinates:  6 , 5 and 11 , 18
Guess X: 10
Guess Y: 12
Guess rectangle area: 65
Your point was inside the rectangle: True
You guessed the correct area!
```
